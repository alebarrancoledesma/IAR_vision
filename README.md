## Visión por Computador Aplicada a la Robótica

### Autores
- Alejandro Barranco Ledesma  
- Eliseo Romero Marín  

---

Proyecto centrado en la evaluación de **YOLO** y otros **sistemas de visión por computador** aplicados a la **robótica**.

---

### Instrucciones de uso

Este proyecto se ejecuta mediante **Jupyter Notebook**. Para comenzar:

1. Abre Jupyter Notebook (o JupyterLab) en el directorio raíz del repositorio.
2. Ejecuta el notebook **`IAR_vision.ipynb`**.

El notebook incluye toda la explicación del flujo de trabajo y los pasos necesarios para la ejecución, por lo que el contenido se considera **autodocumentado** dentro del propio archivo.
